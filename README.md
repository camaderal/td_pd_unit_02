# td_pd_unit_02
Basketball Stats Tool
 - Python Development Tech Degree Unit 02 Project

 Project Details
  - A console-based program that allows a user to view stats of different teams 
  - Data for each team and player are imported from a separate file containing constant data
  - On initialization, each player data are cleaned and assigned to different teams so that each team will have equal number or players and equal number of experienced and inexperienced players
  - Statisctics for each team are calculated and displayed to the user
